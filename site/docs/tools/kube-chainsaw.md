# kube-chainsaw

Graph-level RBAC privilege chain analysis for Kubernetes manifests.

**Full documentation:** [ugiordan.github.io/kube-chainsaw](https://ugiordan.github.io/kube-chainsaw/)

**Repository:** [github.com/ugiordan/kube-chainsaw](https://github.com/ugiordan/kube-chainsaw)

kube-chainsaw is a Go binary and is not bundled in this Action. Use it separately via its own [GitHub Action](https://github.com/ugiordan/kube-chainsaw#github-action).
